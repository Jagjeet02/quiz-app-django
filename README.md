# quiz-app-django
# Django Quiz App

## Features
- Start a new quiz session
- Answer random questions from the database
- View quiz summary (Total, Correct, Incorrect)
- **Enhancements**: Timer, Leaderboard (optional)

## Requirements
- Python 3.x
- Django
- Gunicorn (for production deployment)
